### 相关API
`account`账号数据表:
```javascript
{
    id              # 账号ID
    username        # 用户名（登录使用）
    password        # 账号密码（登录使用）
    role            # 权限组
}
```
#### 1. 添加人员（仅限管理员）
用于添加访问账户，可选用户所属角色组，关系到访问权限。新添加的用户的密码都是`123456`，并在登录时以`is_simple`标记用户是否修改了默认密码。
```javascript
{
    url : http://127.0.0.1:7999/api/accounts/
    method : POST
    data : {
        "username"  # 用户名
        "role"      # 角色(粮油公司0, 教体局1, 学校2)
    }
    return : {
        "msg":     # 消息
        "data":    # 数据
        "code":    # 状态码
    }
}
```
示例：
```javascript
{
    url : http://127.0.0.1:7999/api/accounts/
    method : POST
    data : {
        "username" : "ldzx",
        "role" : 2
    }
    return : {
        "msg": "Created successfully",
        "data": {
            "id": 8,
            "username": "ldzx",
            "is_active": true,
            "role": 2
        },
        "code": 201
    }   
}
```

```javascript
    method : POST
    data : {
        "username" # 用户名
        "password" # 密码

        "code"
    }
}
```

## 商品相关
### 商品API
`goods`商品数据表：
```javascript
{
    id              # 商品ID
    name            # 商品名
    category        # 外键字段，商品种类实例，数据库中为category_id,种类id
    unit            # 外键字段，商品计量单位实例，数据库中为unit_id，单位id

    ori_price       # 商品初始价格，在无可用价格周期时使用
```

#### 2. 添加商品
**权限：仅粮油公司组(`role=0`)**  
添加商品，添加时会传入一个价格，系统会用该价格创建当前周期以及结束日期在现在时间点之后的周期所使用的价格对象，并将价格对象的`status`设置为`0`（未申报），需要和新周期添加时一样，修改并提交该价格。  
<font color=red> 修改了初次添加时的逻辑,修改了部分返回字段 </font>

```javascript
{
    url : http://127.0.0.1:8000/api/goods/
    method : POST
    data : {
        "name"          # 商品名称
        "image"         # 商品图片（非必须）
        "description"   # 商品描述（规格）
        "price"         # 商品价格
        "unit"          # 商品计量单位（注意需要先在数据库中添加计量单位，然后传入对应单位的id）
        "category"      # 商品类别（注意需要先在数据库中添加商品类别，然后传入对应单位的id）
    }
    return : {
        "msg",
        "data":{
            "id"
            "name"
            "image"         
            "description"
            "category"
            "price"
            "unit"
        }，
        "code"
    }
}
```

## 价格相关
### <font color=red>商品价格API</font>
`price`商品价格数据表：
```javascript
{
    id              # 价格ID
    product         # 外键字段，价格对应的商品实例，数据库中为product_id，商品id
    price           # 价格数值
    cycle           # 外键字段，价格对应的价格周期实例，数据库中为cycle_id,周期id
    start_date      # 价格开始时间
    end_date        # 价格结束时间
    status          # 价格状态，共五种
    creater_id      # 报价人ID
    create_time     # 报价申报时间
    reviewer_id     # 审核人ID
    reviewer_time   # 报价审核时间
}
```

现在修改为不由粮油公司组(`role=0`)创建价格，而是教体局组(`role=1`)创建价格周期。当价格周期被创建时，会自动为所有商品创建当前价格周期对应的一个价格对象`price`。`price`的初始状态`status`为`0`，表示为未申报。状态总共有4种，`0：未申报,1：未审核，2：已审核，-1：已拒绝,-99已弃用`。一个`price`对象对应一个`product`商品和一个`cycle`价格周期。

#### 1. 查看所有价格
**权限：仅粮油公司组(`role=0`)**  

查看目前已有的价格对象，可使用`product_id`、`status`、`cycle_id`进行筛选。  
<font color=red> 添加了字段筛选,返回字段变动 </font>
<font color=green> 添加了商品种类id和商品名作为筛选，使用`category_id`和`product_name`。返回字段中添加了周期名字`cycle_name` </font>

```javascript
{
    url : http://127.0.0.1:8000/api/price/
    method : GET
    params: {
        page
        page_size
        product_id      # 商品ID，筛选出某个商品的所有价格对象
        product_name    # 商品名，筛选出某个商品的所有价格对象
        status          # 商品状态，筛选出某个状态下的所有价格对象
        cycle_id        # 周期ID，筛选出某个周期下的所有价格对象
        category_id     # 种类ID，筛选出某个种类下的所有价格对象
    }
    return : {
        "msg"
        "data": {
            "count"
            "next"
            "previous"
            "results": [
                {
                    "id"                    # 价格对象的ID,修改报价时需使用
                    "price"                 # 当前价格
                    "start_date"            # 价格开始日期
                    "end_date"              # 价格结束日期
                    "status"                # 价格状态
                    "creater_id"            # 价格创建者ID
                    "create_time"           # 价格创建时间
                    "reviewer_id"           # 审核人ID
                    "review_time"           # 审核时间
                    "product_id"            # 商品ID
                    "cycle_id"              # 价格周期ID
                    "cycle_name"            # 价格周期名
                    "product_name"          # 商品名称
                    "product_unit"          # 商品单位
                    "product_category"      # 商品类别
                    "product_description"   # 商品描述
                }
            ]
        },
        "code"
    }
}
```
